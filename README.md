## Project install

###### Project install require :

-   Composer. [download](https://getcomposer.org/download/)

```
composer create-project thebachtiarz/laralock
```

## Project setup

```
cd laralock ;
cp .env.example .env ;
php artisan key:generate ;
composer install ;
php artisan migrate ;
```
